## How to Add Custom Presets

### Music Preset
1. Edit `src/ai_content/presets/music.py`
2. Create new preset dataclass:
```python
MY_STYLE = MusicPreset(
    name="my-style",
    prompt="[Your style description with elements]",
    bpm=120,
    mood="descriptive word",
    tags=["tag1", "tag2"],
)
```
3. Add to `MUSIC_PRESETS` dictionary

### Video Preset
1. Edit `src/ai_content/presets/video.py`
2. Create new preset dataclass:
```python
MY_SCENE = VideoPreset(
    name="my-scene",
    prompt="Detailed scene description with camera work",
    aspect_ratio="16:9",
    duration=5,
    style_keywords=["keyword1", "keyword2"],
)
```
3. Add to `VIDEO_PRESETS` dictionary

---

## Preset Selection Tips

1. **Start with presets** - They're optimized for quality results
2. **Match mood to content** - Use preset mood descriptors as guide
3. **Consider BPM** - Faster BPM = more energetic feel
4. **Aspect ratio matters** - 16:9 for YouTube, 9:16 for TikTok/Instagram Stories
5. **Combine creatively** - Use cinematic music with space video, etc.
